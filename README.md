# ₿ Bitcoin Vault Dashboard

A premium Bitcoin portfolio dashboard built with HTML, CSS, and JavaScript.

This project is a modern cryptocurrency dashboard interface featuring a glassmorphism design, responsive layout, animations, and blockchain-inspired portfolio monitoring features.

## ✨ Features

- Premium glassmorphism design
- Responsive mobile layout
- Animated BTC balance counter
- Bitcoin portfolio display
- Wallet information card
- Blockchain Connected indicator
- Copy wallet button
- Toast notifications
- Portfolio statistics
- BTC allocation chart
- Security score
- Network status
- Synchronization time
- Floating Bitcoin animation
- Particle background effects
